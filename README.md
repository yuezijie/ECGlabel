# 生日鹅鸭杀寻礼物

这是一个用 Python 标准库 `tkinter` 做的生日小游戏：玩家控制一只鹅在类似《鹅鸭杀》的飞船房间和走廊中探索，找到藏起来的 10 只鸭子并点击“砍倒”，每次会弹出一个 1-10 的礼物编号，方便你在现实中拿出对应礼物。

## 运行方式

```bash
python birthday_goose_hunt.py
```

## 玩法

- 用 `WASD` 或方向键移动玩家鹅。
- 地图包含舰桥、厨房、货舱、医疗舱、反应堆、礼物仓库等房间，以及控制台、箱子、床、植物、沙发、礼物盒、发光门和地面标识等装饰。
- 玩家只能在房间和走廊构成的可行走区域内移动，墙外区域会被碰撞检测挡住，不能隔墙穿越。
- 不提供小地图，需要靠探索和记忆路线找到所有藏鸭。
- 靠近藏起来的鸭子时，鸭子头顶会显示 `?`；进入可点击范围后会显示 `!`。
- 鼠标左键点击显示 `!` 的鸭子，会弹出礼物编号，鸭子会变成地上的鸡腿。
- 找齐 10 只鸭子后，游戏会显示通关祝福。

## 个性化修改

你可以直接编辑 `birthday_goose_hunt.py`：

- 修改 `ENEMY_COUNT` 和 `make_hidden_geese()` 中的位置来调整礼物数量和藏鸭位置。
- 修改 `ROOMS`、`CORRIDORS`、`PROPS`、`DOORS`、`LIGHTS`、`FLOOR_DECALS` 来调整飞船布局、房间名字、装饰物、灯光和地面标识。
- 修改 `WINDOW_WIDTH`、`WINDOW_HEIGHT`、`WORLD_WIDTH`、`WORLD_HEIGHT` 来调整窗口和地图大小。
- 修改 `messagebox.showinfo()` 中的文字，加入你们之间的昵称、梗或生日祝福。
